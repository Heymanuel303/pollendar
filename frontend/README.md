# frontend (Vue 3 + Vite + Tailwind v4)

Generated in **Phase 0** — see [`../docs/PLAN.md`](../docs/PLAN.md).

This folder is intentionally empty until scaffolding. Phase 0 runs:

```bash
npm create vue@latest frontend     # choose: TypeScript, Router, Pinia
cd frontend
npm install
npm install tailwindcss @tailwindcss/vite      # Tailwind CSS v4 (web-verified)
```

Then add the plugin to `vite.config.ts` and `@import "tailwindcss";` to the CSS
entry — exact steps in [`../docs/PLAN.md`](../docs/PLAN.md). Routes, stores, and
components are defined in [`../docs/DESIGN.md`](../docs/DESIGN.md).
